# Web Scraper Tool

A simple tool to extract product information from e-commerce websites.

## Features
- Fetches product title and price
- Handles basic HTTP errors
- Uses BeautifulSoup for HTML parsing

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python scraper.py <url>
```

## Note
Make sure to check the website's `robots.txt` before scraping.
